# Bienvenides Asadito.js 🥩

En este workshop, vamos a aprender y entender las bases de React, construyendo una App muy sencilla donde nos traeremos gifs de gatos a traves de una API.
Me leiste biens, gifs de gatos 🐈🐈🐈 para luego agregar el que quieras a tu lista de favoritos.

Si estás haciendo este workshop en tu casa, podés ver antes de empezar la
presentación que tiene info bastante útil sobre como empezar en React, como funciona, que es un componente, entre otros conceptos importantes que necesitas saber. Igual son temas que se van a ver en la Intro de este
workshop.

Antes de empezar con todo, que vamos a aprender?

- Components
- JSX
- State
- Props
- Events
- React Hooks
- Fetch API
- Thinking in Components

Y este es un temario con lo que vamos a ver en cada paso del workshop:

1. <Link to="/intro">Intro</Link>: Comenzando un proyecto, partes de un proyecto
en Gatsby, Routing y Lazy loading de imágenes.
2. <Link to="/plugins">Plugins</Link>: Tipos de plugins en Gatsby, cómo utilizarlos.
3. <Link to="/paso-01">Workshop - Paso 1</Link>: Creando un token de Github, definiendo 
nuestra query de GraphQL, instalando y configurando plugin.
4. <Link to="/graphql">GraphQL</Link>: Qué es GraphQL, diferencias con una API REST, GraphiQL.
5. <Link to="/paso-02">Workshop - Paso 2</Link>: Haciendo la query para el plugin de Github, 
manejo en frontend de nuestra data, deploy de nuestro portfolio en Netlify.
6. <Link to="/paso-03">Workshop - Paso 3</Link>: Creando nuestro blog.
