# El State de nuestra APP

En este paso vamos a crear el State de nuestra aplicacion, el [**State**](https://es.reactjs.org/docs/state-and-lifecycle.html) o State es la data que va a persistir en nuestro componente, que a su vez ira cambiando mediante acciones o eventos. En nuestro caso, el GIF de gato proviene de una URL que ira cambiando siempre que le demos click a un button. Por lo tanto, la URL de nuestro GIF tiene que ser un State ya que es un dato que debe actualizarse constantemente y es un dato que tiene que pasarse a los componentes hijos.

Para setear un State, vamos a utiliar un [**Hook**](https://es.reactjs.org/docs/hooks-intro.html) porque nuestro componente es funcional y no de tipo clase. Nos vamos a ubicar en el componente **App** e importamos desde `react` algo llamado `{ useState }`.

Luego, crearemos un State para el GIF URL, como dato sera de tipo **string**. Este `hook` recibe 2 valores, el dato en si y un metodo que actualizara ese dato. La sintaxis quedaria de esta manera: `const [gifUrl, setGifUrl] = useState("")`.

**NOTA**: Dentro de los parentesis de `useState()` podemos pasarle el valor inicial que queramos, ya sea numeros, arrays u objetos.

Como ultimo, vamos a necesitar otro State que se llamara **favorites** que sera un Array que contendra muchos GIF URLS (En nuestro caso, nuestros GIF de Gatos favoritos), para luego renderizar cada uno en pantalla.

La sintaxis seria la siguiente: `const [favorites, setFavorites] = useState([])`.

```jsx
import React, { useState } from "react";
import Title from "./components/Title";
import Card from "./components/Card";
import List from "./components/List";

const App = () => {
  const [gifUrl, setGifUrl] = useState("");
  const [favorites, setFavorites] = useState([]);

  return (
    <div className="main-wrapper">
      <Title />
      <Card />
      <List />
    </div>
  );
};

export default App;

```

Muy bien! para recapitular:

- El State `gifUrl` contendra un string la URL del gatito
- El State `favorites` es un Array que contendra muchos strings con URLS de nuestros gatitos favoritos
